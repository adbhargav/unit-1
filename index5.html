Answers

1. Define a statement in mathematical logic: A statement in logic is a declarative sentence that is either true or false, but not both. Example: "The sky is blue."

2. What is a connective in logic? Logical connectives like AND, OR, and NOT are used to build compound statements by connecting simpler statements.

3. Explain a well-formed formula with an example: A well-formed formula (WFF) is a syntactically correct logical expression, such as (A ∧ B) → C.

4. State the Duality law: Duality law asserts that logical statements remain valid if AND and OR operators are swapped and 1s are interchanged with 0s.

5. Differentiate between equivalence and implication: Equivalence (↔) means two statements have the same truth value. Implication (→) implies that if the first statement is true, the second must be true.

6. Define normal forms and name the types: Normal forms are standardized logical expressions like Conjunctive Normal Form (CNF) and Disjunctive Normal Form (DNF).

7. What is a functionally complete set of connectives? A functionally complete set allows representing any logical expression. NAND and NOR are examples of functionally complete connectives.

8. Explain the purpose of predicate calculus: Predicate calculus extends propositional logic by adding quantifiers and predicates to handle expressions involving variables.

9. Define inference in the context of predicate calculus: Inference is deriving conclusions from premises using rules of logic. Example: If "All humans are mortal" and "Socrates is a human," then "Socrates is mortal."

10. What is a tautology? A tautology is a logical formula that is true in all possible scenarios, such as (A ∨ ¬A), which is always true regardless of A's truth value.
